DAY-7 — INFOSYS SQL NOTESSQL — Structured Query LanguageA language used to interact with databases for storing and managing dataCommon Database SystemsMySQL — widely used open-source databasePostgreSQL — advanced relational databaseSQLite — lightweight embedded databasePurpose of SQLUsed to store, retrieve, update, and delete data inside databases.Commonly used in websites and mobile applications--------------------------------- ActionsCreate a database — to store data structureInsert data — add recordsRead data — fetch recordsUpdate data — modify recordsDelete data — remove records-------------------------------SQL CommandsSELECTUsed to display or read data from tablesExampleselect * from table_nameUPDATEUsed to modify existing data in a tableDELETEUsed to remove records from a tableCreating a Table---------------------------------Example structureCREATE TABLE User (id INT,username VARCHAR(60),password VARCHAR(60));This creates a table named User-------------------------------------Table OperationsDROPDeletes the entire table including structureRENAMEChanges the table nameTRUNCATEDeletes all records but keeps table structure----------------------Operators!= not equal< less thangreater than<= less than or equal= greater than or equalANDBoth conditions must be trueORAt least one condition must be trueNOTReverses a conditionSorting and GroupingORDER BYSorts resultsGROUP BYGroups rows with similar valuesAggregate FunctionsMIN() — smallest valueMAX() — largest valueAVG() — average valueCOUNT() — number of recordsPattern MatchingUsed with LIKE operatorA%Values starting with AExample Apple%aValues ending with aExample Sathiya%a%Values containing aExample BallA%eValues starting with A and ending with eExample AppleJoinsUsed to combine data from multiple tablesINNER JOINReturns matching records from both tablesLEFT JOINReturns all records from left table and matches from rightRIGHT JOINReturns all records from right table and matches from leftCROSS JOINCombines every row of one table with every row of anotherAdvanced OperatorsINChecks if value exists in a listBETWEENChecks value within rangeLIKEPattern matchingIS NULLChecks missing valuesUNIONCombines results without duplicatesUNION ALLCombines results including duplicatesINTERSECTReturns common recordsEXCEPTReturns records present in first but not secondAuthentication Flow (JWT Concept)User logs in with username and passwordServer validates credentialsServer generates a JWT tokenClient stores token in local storage or cookiesClient sends token in request headerServer verifies token and grants accessPython JWT ExampleInstall librarypip install pyjwtExampleimport jwtimport datetimeSECRET_KEY = "mysecretkey"payload = {"user_id": 101,"username": "mastan","exp": datetime.datetime.utcnow() + datetime.timedelta(minutes=30)}token = jwt.encode(payload, SECRET_KEY, algorithm="HS256")print(token)---------------------------------------------------------------
