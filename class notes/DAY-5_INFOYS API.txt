DAY-5 — INFOSYS API NOTESAPI — Application Programming InterfaceA way for different software or apps to talk to each other and exchange data.Frontend payloadData sent from the frontend (UI or client side) to the server.Backend receives this payloadThe server takes that data and processes it.Backend sends responseAfter processing, the server returns output/data back to the frontend.GETUsed to retrieve or read data from an API.POSTUsed to send data to the server, usually inside the request body.DELETEUsed to remove data or resources from the server.GET — to get dataFetches information from the API without changing anything.POST — payload — bodySends input data to the backend for processing or storing.DELETE — removes dataDeletes a specific resource using the API.-------------------------------------------------------------------------------------Flask APILibrary — FlaskA lightweight Python library used to build web APIs FastAPIImports FastAPI module.app = FastAPI()Creates the FastAPI application instance.@app.get()Defines an endpoint that handles GET requests.@app.post()Defines an endpoint that handles POST requests.(project/first endpoint)Represents the URL path where the API is accessed.PostmanA tool used to test APIs by sending requests and checking responses.HTTP RequestStandard protocol request, jsonifyapp = Flask(__name__)# ---------- Home Route ----------@app.route("/")def home():    return "Flask API is running"# ---------- GET API ----------# Example: http://127.0.0.1:5000/hello?name=Mastan@app.route("/hello", methods=["GET"])def hello():    name = request.args.get("name", "Guest")    return jsonify({"message": f"Hello {name}"})# ---------- POST API ----------# Send JSON:# { "num1": 5, "num2": 3 }@app.route("/add", methods=["POST"])def add():    data = request.get_json()    num1 = data["num1"]    num2 = data["num2"]    result = num1 + num2    return jsonify({"result": result})# ---------- Run Server ----------if __name__ == "__main__":    app.run(debug=True)
