# SHAIK-MOHAMMAD-MASTAN-VALI_Receipt-and-invoice-digitizer_TEAMD

🧾 Receipt & Invoice Digitizer
This project is a web-based application that helps users upload and manage their receipts and invoices in a digital format. Instead of manually storing bills, users can upload images or PDFs, and the system extracts important details like vendor name, invoice number, date, and total amount.
The application also provides a simple dashboard where users can view their uploaded receipts and track their spending through basic analytics. It makes invoice management easier, more organized, and accessible anytime.

🚀 My Contribution
I mainly worked on the Admin Dashboard and system monitoring features.
I implemented a dedicated admin panel that is accessible only to admin users after login. This ensures proper role-based access control in the application.
In the admin dashboard, I added:

System overview cards showing total users, total uploads, and processed files
Analytics section with charts to visualize system activity
Upload trend chart to track activity over the last 7 days
Recent upload activity table to monitor latest user actions
Users overview section showing all users and their upload count

I also added functionality where the admin can view all invoices uploaded by a specific user. From there, the admin can check details like vendor, invoice ID, amount, and also open the original uploaded file.
Along with functionality, I worked on improving the UI by organizing the layout, aligning charts properly, and maintaining consistent design across all admin pages.
